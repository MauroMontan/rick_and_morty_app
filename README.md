# rick_and_morty_app

This app use the [Rick and morty API](https://rickandmortyapi.com/).

this app is an example for Hero animations, provider and ListBuilder usage.

## Running this project

- install dependencies
```bash
flutter pub get
```
- run debug mode
```bash
flutter run
```
